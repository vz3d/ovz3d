▎Пример кода для кондуктора

// Параметры кондуктора
$fn = 100; // Увеличение количества сегментов для округлых объектов
diameter = 10; // Диаметр кондуктора
length = 50; // Длина кондуктора
hole_diameter = 6; // Диаметр отверстия для сверла

module drill_guide() {
    // Основная часть кондуктора
    difference() {
        cylinder(h = length, d = diameter, center = false); // Внешняя часть
        translate([0, 0, 0])
            cylinder(h = length + 1, d = hole_diameter, center = false); // Отверстие для сверла
    }
}

// Вызов модуля

▎Объяснение кода

2. Модуль drill_guide():

   • Используется difference() для создания отверстия в цилиндре.

   • cylinder(h = length, d = diameter) создает внешний цилиндр кондуктора.

   • translate([0, 0, 0]) cylinder(h = length + 1, d = hole_diameter) создает отверстие для сверла. Мы добавляем небольшое значение к высоте (например, +1), чтобы гарантировать, что отверстие полностью проходит через весь кондуктор.

3. Вызов модуля: В конце кода мы вызываем модуль drill_guide(), чтобы отобразить кондуктор.

В OpenSCAD можно создать модель фрезы, используя простые геометрические примитивы и операции. Вот пример кода для создания модели фрезы с цилиндрической частью и лезвиями:

module end_mill(diameter, length, flute_length, flute_count) {
    // Параметры
    radius = diameter / 2;
    flute_radius = radius * 0.8; // Радиус канавок

    // Основная часть фрезы
    cylinder(h = length, r = radius, center = true);

    // Создание канавок
    for (i = [0 : flute_count - 1]) {
        angle = 360 / flute_count * i;
        translate([0, 0, length / 2])
            rotate([0, 0, angle])
                difference() {
                    // Создаем канавку
                    translate([0, flute_radius, 0])
                        cube([length, flute_radius * 2, flute_length], center = true);
                    // Убираем часть цилиндра
                    translate([0, 0, -flute_length])
                        cylinder(h = flute_length * 2, r = radius + 1, center = true);
                }
    }
}

// Пример использования
end_mill(6, 40, 20, 4);


▎Параметры функции:
- diameter: Диаметр фрезы.
- length: Общая длина фрезы.
- flute_length: Длина канавок (лезвий).
- flute_count: Количество канавок.

Этот код создает простую модель фрезы с заданными параметрами. Вы можете изменять значения при вызове функции end_mill, чтобы получить нужный вам размер и количество лезвий.

В OpenSCAD можно создать зенкер (конусное отверстие) с помощью параметрических функций. Зенкер обычно представляет собой конусное отверстие, которое расширяется к верхней части. Вот пример кода, который демонстрирует, как создать зенкер в OpenSCAD:

module countersink(diameter, depth, angle) {
    // Параметры
    angle_rad = angle * 0.5; // Угол в радианах
    base_radius = diameter / 2;
    top_radius = base_radius + 1; // Увеличиваем радиус для создания скошенной части

    // Создание зенкера
    difference() {
        // Создаем конусное отверстие
        linear_extrude(height = depth) {
            polygon(points = [
                [0, 0],
                [base_radius * cos(angle_rad), base_radius * sin(angle_rad)],
                [top_radius * cos(angle_rad), top_radius * sin(angle_rad)]
            ]);
        }
        
        // Создаем цилиндр для вычитания
        translate([0, 0, -depth])
            cylinder(h = depth, r = base_radius, center = true);
    }
}

// Пример использования
countersink(10, 5, 60);


▎Параметры функции:
- diameter: Диаметр зенкера.
- depth: Глубина зенкера.
- angle: Угол наклона стенки зенкера.

Этот код создает зенкер с заданными параметрами. Вы можете изменять значения при вызове функции countersink, чтобы получить нужный вам размер и угол.

Чтобы выдавить символ радиации в OpenSCAD, вы можете использовать функцию linear_extrude(). Вот как можно модифицировать предыдущий код для создания 3D-объекта:

module radiation_symbol() {
    // Основной круг
    difference() {
        linear_extrude(height = 5) {
            color("yellow") {
                circle(30); // Внешний круг
            }
        }
        linear_extrude(height = 5) {
            color("yellow") {
                circle(20); // Внутренний круг
            }
        }
    }

    // Три "луча"
    for (angle = [0, 120, 240]) {
        rotate(angle) {
            translate([0, 20, 0]) {
                color("black") {
                    rotate(45) {
                        linear_extrude(height = 5) {
                            polygon(points=[[0, 0], [10, 30], [-10, 30]]);
                        }
                    }
                }
            }
        }
    }
}

radiation_symbol();


▎Объяснение изменений:
- Я добавил linear_extrude(height = 5) вокруг каждого элемента, который вы хотите выдавить. Параметр height определяет высоту выдавливания.
- Теперь весь объект будет трехмерным с высотой 5 единиц.

Скопируйте этот код в OpenSCAD и выполните его, чтобы увидеть 3D-символ радиации. Вы можете изменять значение height, чтобы изменить высоту выдавливания.

Создание направляющей для сверла (кондуктора) с использованием CadQuery — это отличный способ продемонстрировать возможности этой библиотеки для параметрического моделирования. Ниже приведен пример кода, который создает простую направляющую для сверла.

▎Пример: Направляющая для сверла

import cadquery as cq

# Параметры кондуктора
diameter = 10  # Диаметр направляющей
length = 50    # Длина направляющей
hole_diameter = 5  # Диаметр отверстия для сверла
hole_depth = 40    # Глубина отверстия

# Создание основного цилиндра (направляющей)
conductor = (
    cq.Workplane("XY")
    .circle(diameter / 2)  # Создаем круг с заданным диаметром
    .extrude(length)        # Экструзия в длину
)

# Создание отверстия для сверла
conductor = (
    conductor.faces(">Z")  # Выбираем верхнюю грань
    .workplane()           # Переходим на рабочую плоскость
    .hole(hole_diameter, hole_depth)  # Создаем отверстие
)

# Экспорт модели в STL файл
conductor.exportStl("drill_conductor.stl")

# Для визуализации в Jupyter Notebook (если необходимо)
show_object(conductor)


▎Описание кода:

1. Импорт библиотеки: Импортируем CadQuery.

2. Определение параметров: Задаем параметры, такие как диаметр направляющей, длина, диаметр и глубина отверстия.

3. Создание цилиндра: Используем circle() для создания основания и extrude() для экструзии в длину, чтобы получить основной цилиндр.

4. Создание отверстия: Переходим на верхнюю грань цилиндра с помощью faces(">Z") и создаем отверстие с помощью метода hole().

5. Экспорт модели: Экспортируем созданную модель в файл формата STL для дальнейшего использования или печати.

6. Визуализация: В конце можно визуализировать объект в Jupyter Notebook.

▎Заключение

Этот пример демонстрирует, как легко можно создать простую направляющую для сверла с помощью CadQuery. Вы можете изменять параметры, чтобы адаптировать модель под свои нужды или добавлять дополнительные функции, такие как ручки или другие элементы дизайна.

• Пересечение: intersect()

result = cube.cut(cq.Workplane("XY").cylinder(5, 10))

CadQuery — это библиотека для параметрического 3D-моделирования, написанная на Python. Она предоставляет простой и интуитивно понятный интерфейс для создания сложных 3D-объектов с использованием кода. CadQuery позволяет пользователям создавать модели, используя логические операции, такие как объединение, вычитание и пересечение, что делает его мощным инструментом для автоматизации процесса проектирования.

▎Основные особенности CadQuery:

1. Параметрическое моделирование: Вы можете легко изменять размеры и параметры объектов, что позволяет быстро генерировать различные варианты моделей.

2. Читаемость кода: CadQuery стремится быть понятным и доступным, что делает его удобным для тех, кто не является профессиональным программистом.

3. Интеграция с CAD-системами: Модели, созданные в CadQuery, могут быть экспортированы в форматы, совместимые с различными CAD-системами, такими как STEP и STL.

4. Мощные геометрические операции: CadQuery поддерживает множество геометрических операций, таких как создание эскизов, выдавливание, вращение и т.д.

5. Поддержка визуализации: Библиотека может быть использована совместно с другими инструментами для визуализации моделей, что позволяет пользователям видеть результаты своей работы в реальном времени.

▎Пример использования:

Вот простой пример кода на Python с использованием CadQuery для создания куба:

import cadquery as cq

# Создание куба размером 10x10x10
result = cq.Workplane("XY").box(10, 10, 10)

# Экспорт модели в формат STL
result.exportStl("cube.stl")


CadQuery активно используется в различных областях, включая инженерное проектирование, архитектуру и прототипирование. Это делает его отличным инструментом для разработчиков и дизайнеров, которые хотят автоматизировать процесс создания 3D-моделей.

OpenSCAD — это открытая система автоматизированного проектирования (САПР) для параметрического создания твердотельных трёхмерных объектов.

Особенности программы:

не является интерактивным редактором;

строит модель на основе текстового описания и перестраивает её при каждом его изменении;

предоставляет два основных метода моделирования: конструктивная блочная геометрия и экструзия (выдавливание) двухмерных контуров;

для задания двухмерных контуров используются AutoCAD DXF-файлы;

может читать и создавать 3D-модели в форматах файлов DXF, STL и OFF.

OpenSCAD доступна для следующих операционных систем: Windows, Linux, macOS, FreeBSD, OpenBSD и NetBSD.
