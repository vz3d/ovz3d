Создание кондуктора (направляющей) для сверла в OpenSCAD — это отличный проект, который поможет вам освоить основные конструкции и трансформации. Кондуктор обычно имеет цилиндрическую форму, чтобы направлять сверло, и может включать дополнительные элементы для стабилизации. Вот пример простого кода для создания кондуктора для сверла.

▎Пример кода для кондуктора

// Параметры кондуктора
$fn = 100; // Увеличение количества сегментов для округлых объектов
diameter = 10; // Диаметр кондуктора
length = 50; // Длина кондуктора
hole_diameter = 6; // Диаметр отверстия для сверла

module drill_guide() {
    // Основная часть кондуктора
    difference() {
        cylinder(h = length, d = diameter, center = false); // Внешняя часть
        translate([0, 0, 0])
            cylinder(h = length + 1, d = hole_diameter, center = false); // Отверстие для сверла
    }
}

// Вызов модуля
drill_guide();


▎Объяснение кода

1. Параметры: 

   • $fn = 100; — это количество сегментов, используемых для округлых объектов, что делает их более гладкими.

   • diameter — диаметр внешней части кондуктора.

   • length — длина кондуктора.

   • hole_diameter — диаметр отверстия, в которое будет вставлено сверло.

2. Модуль drill_guide():

   • Используется difference() для создания отверстия в цилиндре.

   • cylinder(h = length, d = diameter) создает внешний цилиндр кондуктора.

   • translate([0, 0, 0]) cylinder(h = length + 1, d = hole_diameter) создает отверстие для сверла. Мы добавляем небольшое значение к высоте (например, +1), чтобы гарантировать, что отверстие полностью проходит через весь кондуктор.

3. Вызов модуля: В конце кода мы вызываем модуль drill_guide(), чтобы отобразить кондуктор.

▎Настройка

Вы можете настроить параметры (диаметры и длину) в зависимости от вашего сверла и потребностей проекта. Если вам нужно добавить дополнительные элементы, такие как ручки или крепления, вы можете сделать это с помощью дополнительных примитивов и трансформаций.

▎Основные элементы OpenSCAD

1. Простые примитивы

   • cube(size): создает куб с заданным размером.

   • sphere(radius): создает сферу с заданным радиусом.

   • cylinder(h, r): создает цилиндр с высотой h и радиусом r.

   • polyhedron(points, faces): создает произвольный полигон по заданным вершинам и граням.

2. Трансформации

   • translate([x, y, z]): перемещает объект по координатам.

   • rotate([x, y, z]): вращает объект вокруг заданных осей.

   • scale([x, y, z]): изменяет размер объекта.

3. Композиция объектов

   • union(): объединяет несколько объектов в один.

   • difference(): вычитает один объект из другого.

   • intersection(): создает новый объект из пересечения двух объектов.

4. Модули

   • Вы можете создавать модули для организации кода:
          module my_shape() {
         cube(10);
         translate([5, 5, 0]) sphere(5);
     }
     my_shape();
     

5. Параметры

   • Вы можете задавать параметры для модулей:
          module my_shape(size) {
         cube(size);
     }
     my_shape(10);

     }

В OpenSCAD можно создать модель фрезы, используя простые геометрические примитивы и операции. Вот пример кода для создания модели фрезы с цилиндрической частью и лезвиями:

module end_mill(diameter, length, flute_length, flute_count) {
    // Параметры
    radius = diameter / 2;
    flute_radius = radius * 0.8; // Радиус канавок

    // Основная часть фрезы
    cylinder(h = length, r = radius, center = true);

    // Создание канавок
    for (i = [0 : flute_count - 1]) {
        angle = 360 / flute_count * i;
        translate([0, 0, length / 2])
            rotate([0, 0, angle])
                difference() {
                    // Создаем канавку
                    translate([0, flute_radius, 0])
                        cube([length, flute_radius * 2, flute_length], center = true);
                    // Убираем часть цилиндра
                    translate([0, 0, -flute_length])
                        cylinder(h = flute_length * 2, r = radius + 1, center = true);
                }
    }
}

// Пример использования
end_mill(6, 40, 20, 4);


▎Параметры функции:
- diameter: Диаметр фрезы.
- length: Общая длина фрезы.
- flute_length: Длина канавок (лезвий).
- flute_count: Количество канавок.

Этот код создает простую модель фрезы с заданными параметрами. Вы можете изменять значения при вызове функции end_mill, чтобы получить нужный вам размер и количество лезвий.

В OpenSCAD можно создать зенкер (конусное отверстие) с помощью параметрических функций. Зенкер обычно представляет собой конусное отверстие, которое расширяется к верхней части. Вот пример кода, который демонстрирует, как создать зенкер в OpenSCAD:

module countersink(diameter, depth, angle) {
    // Параметры
    angle_rad = angle * 0.5; // Угол в радианах
    base_radius = diameter / 2;
    top_radius = base_radius + 1; // Увеличиваем радиус для создания скошенной части

    // Создание зенкера
    difference() {
        // Создаем конусное отверстие
        linear_extrude(height = depth) {
            polygon(points = [
                [0, 0],
                [base_radius * cos(angle_rad), base_radius * sin(angle_rad)],
                [top_radius * cos(angle_rad), top_radius * sin(angle_rad)]
            ]);
        }
        
        // Создаем цилиндр для вычитания
        translate([0, 0, -depth])
            cylinder(h = depth, r = base_radius, center = true);
    }
}

// Пример использования
countersink(10, 5, 60);


▎Параметры функции:
- diameter: Диаметр зенкера.
- depth: Глубина зенкера.
- angle: Угол наклона стенки зенкера.

Этот код создает зенкер с заданными параметрами. Вы можете изменять значения при вызове функции countersink, чтобы получить нужный вам размер и угол.

Чтобы выдавить символ радиации в OpenSCAD, вы можете использовать функцию linear_extrude(). Вот как можно модифицировать предыдущий код для создания 3D-объекта:

module radiation_symbol() {
    // Основной круг
    difference() {
        linear_extrude(height = 5) {
            color("yellow") {
                circle(30); // Внешний круг
            }
        }
        linear_extrude(height = 5) {
            color("yellow") {
                circle(20); // Внутренний круг
            }
        }
    }

    // Три "луча"
    for (angle = [0, 120, 240]) {
        rotate(angle) {
            translate([0, 20, 0]) {
                color("black") {
                    rotate(45) {
                        linear_extrude(height = 5) {
                            polygon(points=[[0, 0], [10, 30], [-10, 30]]);
                        }
                    }
                }
            }
        }
    }
}

radiation_symbol();


▎Объяснение изменений:
- Я добавил linear_extrude(height = 5) вокруг каждого элемента, который вы хотите выдавить. Параметр height определяет высоту выдавливания.
- Теперь весь объект будет трехмерным с высотой 5 единиц.

Скопируйте этот код в OpenSCAD и выполните его, чтобы увидеть 3D-символ радиации. Вы можете изменять значение height, чтобы изменить высоту выдавливания.

В OpenSCAD вы можете создать символ радиации, используя комбинацию примитивов и трансформаций. Вот пример кода, который создает знак радиации:

module radiation_symbol() {
    // Основной круг
    color("yellow") {
        difference() {
            circle(30); // Внешний круг
            circle(20); // Внутренний круг
        }
    }

    // Три "луча"
    for (angle = [0, 120, 240]) {
        rotate(angle) {
            translate([0, 20, 0]) {
                color("black") {
                    rotate(45) {
                        polygon(points=[[0, 0], [10, 30], [-10, 30]]);
                    }
                }
            }
        }
    }
}

radiation_symbol();


▎Объяснение кода:
- Модуль radiation_symbol() создает знак радиации.
- Используется circle() для создания внешнего и внутреннего кругов.
- difference() удаляет внутренний круг из внешнего, создавая кольцо.
- В цикле for создаются три луча, которые располагаются под углом 120 градусов друг к другу.
- polygon() используется для создания формы луча.

Вы можете изменять размеры и цвета по своему усмотрению. Скопируйте этот код в OpenSCAD, чтобы увидеть результат.
